Introduction To Scala

Scala (pronounced as Scaaah-lah) is multi purpose programming language targeting the JVM (That compiles to bytecode) and tries to combine both object oriented and functional paradigms of programming. Scala was created by Dr.Martin Odersky and the official website of scala is www.scala-lang.org. As per the official website, scala is

Object Oriented Meets Functional. Have the best of both the worlds. Construct elegant class hierarchies for maximum code reuse and extensibility, implement their behaviour using higher-order functions. Or anything in-between.

Scala is a beautiful language that provides the programmer with lots of powerful abstractions to program. Since it compiles to bytecode, it can seamlessly interop with other Java and other JVM based languages.

Statically Typed

Scala is a statically typed programming language. Verbosity is usually one of the arguments against a statically typed programming language. But scala relieves a lot of that pain by inferring the types whenever possible. Also scala provides more syntactic advantages over other statically typed programming languages. Scala has a very complex type system that may seem daunting at first. But the reality is if you are a person using scala for general programming, you can write beautiful code even without knowing the type system completely. But as it is said, if you want to achieve greatness, you have to toil hard. If you want to reap the benefits of scala and fall in love with your code completely, you have to put in effort to master its type system.

Scala as a scripting language

Scala can also be used as a scripting language. Just enter some scala code, save it in a file with the extension as ".scala", execute the command "scala filename.scala".

Web development in scala

Scala has lots of powerful frameworks for creating web applications. Play is the defacto web framework for Scala backed by TypeSafe. TypeSafe is the commercial entity formed for providing professional support to Scala and its frameworks.
